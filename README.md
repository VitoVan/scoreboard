# Score Board
A board to record scores.

Usage: **Left click to +1, right click to -1**, edit [this line](https://github.com/VitoVan/scoreboard/blob/15050136e91966ccde13ba4caebce5cbf5391223/canvas.lisp#L4) to change names.
